# Property CRUD API:

## Backend:

- Create property table (id, title, price, status) using Drizzle ORM and MySQL.
- Build REST API using Express + Postman.
- Validate inputs using zod or express-validator.

## Postman:

- Create full API collection.
- Write tests for success + failure cases.

## Installation

```
# Install dependencies

npm install

# Start development server

node index.ts
```
